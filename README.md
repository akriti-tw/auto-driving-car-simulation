# Auto Driving Car Simulation – Run Guide

This document explains how to build and run the application locally.

## Prerequisites

- **Java 17 or higher**

Verify Java installation:

```bash
 java -version
```

## Run the Application

```bash
 ./gradlew build
```

After building, run the application with:

```bash
 java -cp build/classes/java/main org.Main
```
